# Quiet Bloom (Android Period & Wellbeing Tracker)

## Pages with further information and user study
https://tschnu.github.io/CCL3-WS2025/

## Brief description
Quiet Bloom is an offline-first Android app for tracking menstrual cycles, mood, energy, pain, and journal entries. Using data from previous months, the app predicts upcoming periods and ovulation and visualizes wellbeing patterns.

## App Goal
Provide a simple and calm period-tracking app that helps users better understand how emotional and physical wellbeing changes across the menstrual cycle while offering clear predictions and visual insights.

## Core features
- Full CRUD for daily entries  
- Period and ovulation prediction  
- Calendar overview (past and future months)  
- Multiselect period logging  
- Statistics & custom graphs  
- Journal entries  
- Offline-first design  
- Local database storage  

## Tech stack
- Kotlin  
- Jetpack Compose  
- Room (SQLite)  
- Offline-first architecture

## Build & Run

### Requirements
- Android Studio (latest stable)  
- Android SDK 24+

### Download App
- go to pages (https://tschnu.github.io/CCL3-WS2025/) and download apk file nto device

### Run
1. Clone repository  
2. Open `code/` in Android Studio  
3. Let Gradle sync  
4. Run on emulator or physical device
